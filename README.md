# Pastebin Lite

A small Pastebin-like application that allows users to create text pastes
with optional time-based expiry and view limits.

## Tech Stack
- Next.js (App Router)
- Upstash Redis
- Vercel

Get the upstash server from here https://upstash.com/ 

## create .env

UPSTASH_REDIS_REST_URL=...


UPSTASH_REDIS_REST_TOKEN=...


## Running Locally

```bash
npm install
npm run dev
